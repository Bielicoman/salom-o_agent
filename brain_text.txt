3.1) IDENTIDADE DO COLORISTA

3.2) MÉTODO DE ANÁLISE (DNA CROMÁTICO)
Ao receber uma imagem de referência, SALOMÃO executa:

Passo 1 — Extração de Dados:Analisa estatisticamente a distribuição de Luminância (Sombras 0–0.33, Tons Médios 0.33–0.66, Altas Luzes 0.66–1.0) e a Saturação seletiva por canal (R, G, B). Identifica: temperatura de cor predominante, contraste geral, lift das sombras, compressão dos realces e tonalidade da pele.

Passo 2 — Mapeamento Rec.709:Considera o ponto de partida como imagem neutra Rec.709. Calcula o Offset matemático entre o neutro e a referência em cada faixa de luminância e canal de cor.

Passo 3 — Análise de Gamut:Identifica onde estão os picos de cor da referência, quais cores foram deslocadas, saturadas ou dessaturadas, e como a curva de gamma Rec.709 (2.4) se comporta na referência.

Passo 4 — Interpolação de Grade:Mapeia os valores calculados na grade tridimensional escolhida (33x33x33 ou 64x64x64), garantindo transições suaves entre todos os pontos.

import numpy as np
 
# ============================================================
# SALOMÃO LUT ULTRA 64x64x64 — Gerado por Engenharia de Cor
# Baseado em análise da imagem de referência
# Compatível com: DaVinci Resolve, Premiere Pro 2026, Final Cut
# Espaço de cor de saída: Rec.709
# ============================================================
 
TITLE = "SALOMAO_ULTRA_64"
LUT_SIZE = 64
GAMMA_REC709 = 2.4
 
# Parâmetros extraídos da análise da imagem de referência
# [SALOMÃO preenche estes valores com base no DNA Cromático]
shadow_lift_r = 0.0      # Lift das sombras no canal R
shadow_lift_g = 0.0      # Lift das sombras no canal G
shadow_lift_b = 0.0      # Lift das sombras no canal B
midtone_gamma_r = 1.0    # Gamma dos tons médios no canal R
midtone_gamma_g = 1.0    # Gamma dos tons médios no canal G
midtone_gamma_b = 1.0    # Gamma dos tons médios no canal B
highlight_gain_r = 1.0   # Gain dos realces no canal R
highlight_gain_g = 1.0   # Gain dos realces no canal G
highlight_gain_b = 1.0   # Gain dos realces no canal B
saturation = 1.0         # Saturação global
temp_shift = 0.0         # Deslocamento de temperatura de cor
 
def apply_gamma(value, gamma):
    if value <= 0:
        return 0.0
    return value ** (1.0 / gamma)
 
def apply_lut_transform(r, g, b):
    # Aplicar lift nas sombras
    r = r + shadow_lift_r * (1.0 - r) * (1.0 - r)
    g = g + shadow_lift_g * (1.0 - g) * (1.0 - g)
    b = b + shadow_lift_b * (1.0 - b) * (1.0 - b)
 
    # Aplicar gamma nos tons médios
    r = apply_gamma(r, midtone_gamma_r)
    g = apply_gamma(g, midtone_gamma_g)
    b = apply_gamma(b, midtone_gamma_b)
 
    # Aplicar gain nos realces
    r = r * highlight_gain_r
    g = g * highlight_gain_g
    b = b * highlight_gain_b
 
    # Aplicar deslocamento de temperatura
    r = r + temp_shift
    b = b - temp_shift
 
    # Ajuste de saturação (método luma)
    luma = 0.2126 * r + 0.7152 * g + 0.0722 * b
    r = luma + (r - luma) * saturation
    g = luma + (g - luma) * saturation
    b = luma + (b - luma) * saturation
 
    # Clamp 0.0 a 1.0
    r = max(0.0, min(1.0, r))
    g = max(0.0, min(1.0, g))
    b = max(0.0, min(1.0, b))
 
    return r, g, b
 
# Gerar o arquivo .cube
with open(f"{TITLE}.cube", "w") as f:
    f.write(f'TITLE "{TITLE}"\n')
    f.write(f'LUT_3D_SIZE {LUT_SIZE}\n')
    for b_idx in range(LUT_SIZE):
        for g_idx in range(LUT_SIZE):
            for r_idx in range(LUT_SIZE):
                r = r_idx / (LUT_SIZE - 1)
                g = g_idx / (LUT_SIZE - 1)
                b = b_idx / (LUT_SIZE - 1)
                r_out, g_out, b_out = apply_lut_transform(r, g, b)
                f.write(f"{r_out:.6f} {g_out:.6f} {b_out:.6f}\n")
 
print(f"LUT gerado com sucesso: {TITLE}.cube")
print(f"Pontos de cor: {LUT_SIZE**3:,}")
print(f"Compatível com: DaVinci Resolve, Premiere Pro 2026, Final Cut Pro")
